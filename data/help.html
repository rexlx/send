rxlx GitHub $ send -h Usage of send: -c string specify command -conf string json config location -fatal return failed exit codes -file string specify a file separated by newline -hosts string multiple hosts inside quotes -key string path to key if not default -log PROTO@ADDR:PORT for flat file: /path/name.ext, syslog: PROTO@ADDR:PORT -ordered run in order instead of async -port int port number (default 22) -timeout int timeout in seconds (default 90) -user string username
